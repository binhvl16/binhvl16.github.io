Rounded Corners

The .img-rounded class add rounded corners to an image (not available in IE8)

Circle

The .img-circle class shapes the image to a circle (not available in IE8).

Thumbnail

The .img-thumnail class shapes the image to a thumbnail.

Responsive Images

Responsive images automatically adjust to fit the size of the screen.

Create responsive images by adding an .img-responsive class to the img tag. The image will then scale nicely to the parent element.
